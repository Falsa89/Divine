# Divine Waifus — Emergent Handoff Pack

Version: 0.1

## Purpose

This pack contains readable project docs and structured JSON data so Emergent can stop relying on long chat summaries.

Recommended repo paths:

- `docs/divine/*`
- `data/design/*`

## How to use with Emergent

Ask Emergent to read only the file(s) relevant to the current task.

Example:

```text
Read docs/divine/01_DIVINE_WAIFUS_WORKING_BIBLE.md and data/design/heroes_master.json.
Do not modify DB.
Prepare a dry-run plan only.
```

## Important

Do not “apply everything” from this pack at once.  
Use it as source documentation and structured data for future tasks.
